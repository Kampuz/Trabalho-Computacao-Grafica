# Trabalho Parte 2 de Computação Gráfica

Todas as funcionalidades que deve ter o trabalho. Evitando mencionar que algumas das implementações anteriores são necessárias para as implementações das funcionalidades a seguir.

- [ ] Aula 17 : Implemente o algoritmo da memória de profundidade (Z-buffer), com alguns objetos 3D. Apresente usando a projeção ortogonal (z=0), permita ao usuário rotacionar os objetos em torno da origem, como no trabalho da casinha (cor de fundo = preto)
- [ ] Aula 18 e 19 Prática 1 : Implementar as superfícier bilineares juntamente com o Z-buffer e as transformações geométricas, para exibir interativamente (rotacionar e exibir) os 7 objetos
- [ ] Aula 18 e 19 Prática 2 : Implementar um ambiente para criar objetos 2D usando varredura rotacional. O usuário deve desenhar uma curva (perfil), manualmente, ao lado de um eixo e, em seguida, o objeto é construído.
